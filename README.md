# Official Repository for HayabusaNet

Thank you for your interest in our work.

This repository has been created as the official implementation page for **HayabusaNet**, a lightweight hybrid attention-based multiscale fusion CNN for brain tumor classification using MRI images.

The related manuscript is currently under review. The source code, models, and documentation will be released after the manuscript has been accepted/published, so that the public implementation is consistent with the final version of the paper.

During the review process, the materials may be shared by the corresponding author upon reasonable request.

Please watch this repository or star ⭐ it for future updates.
